My First Contact with HTML & CSS

In my first attempt learning how to code, I get contact with HTML & CSS. Here I learn about the basic HTML CSS Structure & setup as well as learning how to create basic projects.

How long did it take?

The module itself span for 6.3 hours of learning materials. If I remember correctly, it took me about 2-3 weeks as a first time code learner. It was fun! I get to do several mini projects and a Solo Project called Hometown Homepage.

The module consists of 4 mini projects & 1 solo project. I learn the basic of HTML & CSS Structure. Learned how to make an interactive website using pseudo hover. How to make an image background scale to fit the width of its container. Custom font, basic flexbox skills and so on...

Additionally, I've also taken additional HTML CSS course when I was stuck learning JavaScript on later time which adds another excercise project to my coding portfolio.

Care to tell a little bit about your Hometown Homepage Solo Project?

For this Hometown Homepage Solo Project, I choose to present Bali as my chosen city. The first time I went to Bali I was stunned with how sacred the place feels, the peace and calmness when I visited Pura Uluwatu Temple and the melodramatic feeling when I watch sunset by the beach. As if time itself was frozen in place and nothing else matters. I was also mesmerized by the people culture and their local Cuisine. Babi Guling was definitely one of my many favorite.
